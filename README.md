# Library Management System :
This is a mini project website developed using PHP and MySQL.

# Description :
The Library Management System helps to manage books in a library.  
It allows the admin to log in, add new books, and issue books to students.

# Technologies Used :
- PHP
- MySQL
- HTML
- CSS

# Features 
- Admin Login
- Add Book
- Issue Book
- Logout

# How to Run the Project :
1. Install XAMPP
2. Start Apache and MySQL
3. Import the database using phpMyAdmin
4. Place the project folder inside 'htdocs'
5. Open browser and run the project on localhost

# Project Type :
Mini Project
